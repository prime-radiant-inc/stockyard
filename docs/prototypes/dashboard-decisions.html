7. VM Actions & Controls

How should stop/destroy/snapshot actions be presented?

Inline Buttons

Action buttons directly on each VM row/card. Fast access, always visible.

vm-abc123
running
SSH Snap Stop

Hover Reveal (Recommended)

Actions appear on hover. Cleaner default view, actions when you need them.

vm-abc123
running
jesse · 2h
vm-def456
running
SSH Snap Stop
Recommended: Reduces visual clutter. Actions appear exactly when you're focused on a VM.

Dropdown Menu

Three-dot menu revealing all actions. Most compact, works for many actions.

Actions only on VM detail view. Forces deliberate navigation before action.
